Design notes

I struggled a bit with setting up the design to match the original sketch,
but I feel like what I've created here is a natural compromise.
My main goal with this portfolio is to provide information in a way that's "frills-free."
While I put a lot of effort into the color and font choice,
I intentionally wanted to choose a design that does not obscure information.
Clarity is the number one goal.

Moving forwards

I'd like to potentially add a shrunken down navbar for the mobile version of the page.
In addition, some general formatting fixes would go a long way.
Of course, actually adding in portfoio content is also a priority,
in addition to potentially adding more portfolio spaces for additional projects.
